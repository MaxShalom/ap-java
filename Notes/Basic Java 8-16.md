### For Loops

```java
for(int i = 0; i < 10; i++)
{
  // Code here will execute 10 times.
}

for(int i = 0; i > 10; i--)
{
  System.out.println(i);
  // Code here will print the number 10 and cunt down to 0.
}

for(int i = 0; i <= 100; i += 2)
{
  System.out.println(i);
  // Code here will count by 2s till 100.
}
```
